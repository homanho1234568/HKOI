HKOI 必背重點

考前黃金 30 分鐘快速複習：常見陷阱、演算法思路與比賽小技巧。

核心公式：輸入 → 處理 → 輸出

九成題目都符合這個流程。保持程式結構乾淨可讀，除錯才能省時間。

# 1. 讀入
n = int(input())
arr = [int(x) for x in input().split()]

# 2. 處理
arr.sort()
answer = arr[0] + arr[-1]

# 3. 輸出
print(answer)

Python 三大陷阱

  1. 輸入型別：記得用 int()、float()。不要拿字串直接做加法。
  2. 整除與浮點：/ 會得到浮點數。若題目要求整數，請改用 //。
  3. Print 空格：print(a, b) 會插入空格，可用 print(f"{a}{b}") 控制格式。

演算法起手式

  • 不確定策略時，先排序再觀察。許多貪心或搜尋題都以排序開局。
  • 需要查找元素時先想：是否可以用 set 或 dict 提升效率。
  • 遇到「最大／最小」且帶限制的問題，試著用二分搜尋或前綴和。

考場小撇步

  • 準備模板：快速輸入輸出、排序、最大公因數、幾何距離。
  • 分類題型：選擇題先排除明顯錯誤；程式題先寫最簡子任務。
  • 確保每題輸出後都馬上測試 Sample Input，最後再整體交叉檢查。

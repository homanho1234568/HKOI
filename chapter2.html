第二章｜流程控制與決策

If/Else 判斷與迴圈結構是演算法的骨架，本章整理經典模式與 HKOI 常見應用。

If / Else 分支

條件判斷是將題目規則轉成程式碼的第一步，可以使用巢狀結構或提早返回，讓邏輯更清楚。

score = int(input())

if score >= 90:
    print("金牌")
elif score >= 75:
    print("銀牌")
elif score >= 60:
    print("銅牌")
else:
    print("繼續努力")
技巧：先處理條件最嚴格或最簡單的情況，可減少巢狀深度與重複判斷。

While 與 For 迴圈的選擇

For 迴圈

適合已知迭代次數的情況，例如讀取固定筆數的輸入或枚舉範圍。

total = 0
for _ in range(5):
    total += int(input())
print(total)

While 迴圈

條件式不固定或需要逐步逼近答案時使用，例如模擬或直到某狀態為止。

n = int(input())
step = 0
while n > 1:
    if n % 2 == 0:
        n //= 2
    else:
        n = 3 * n + 1
    step += 1
print(step)

綜合範例：最大連續區段

題目：輸入一串 0/1 序列，求連續 1 最長長度。

s = input().strip()
best = 0
current = 0

for ch in s:
    if ch == "1":
        current += 1
        best = max(best, current)
    else:
        current = 0

print(best)

此題練習了迴圈、條件判斷與變數重設，是初級組常見題型。

練習題

  1. 輸入整數 n，輸出 1 到 n 之間所有偶數的平方和。
  2. 給一串整數，請偵測是否為非遞減序列（每個數字不小於前一項）。
  3. 模擬石頭遊戲：兩人輪流取 1 或 2 顆，請判斷誰會勝出（假設雙方皆完美策略）。

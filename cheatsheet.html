Python 語法速查表

考前最後五分鐘，用這份表快速回顧輸入輸出、常用運算符與邏輯語法。

輸入與輸出

語法 說明 範例
input() 讀入一行字串 name = input()
int() 把字串轉成整數 a = int(input())
map(int, input().split()) 空白分隔多個整數 a, b = map(int, input().split())
print() 輸出，預設結尾為換行 print("Ans:", value)
print(value, end="") 控制輸出結尾，避免多餘換行 print(x, end=" ")

常用運算符

符號 名稱 範例（a=10, b=3） 結果
+ 加法 a + b 13
- 減法 a - b 7
* 乘法 a * b 30
/ 除法（浮點數） a / b 3.333...
// 整除 a // b 3
% 取餘數 a % b 1
** 次方 a ** b 1000

比較與布林運算

語法 說明 範例
== 等於 a == b
!= 不等於 a != b
>, < 大於、小於 a > b
>=, <= 大於等於、小於等於 a >= b
and 布林 AND (a > 0) and (b > 0)
or 布林 OR (a > 0) or (b > 0)
not 布林 NOT not (a == b)

常用技巧片段

# 讀入 n 與串列
n = int(input())
arr = list(map(int, input().split()))

# 檢查是否為遞增
is_sorted = all(arr[i] <= arr[i + 1] for i in range(len(arr) - 1))

# 前綴和
prefix = [0]
for value in arr:
    prefix.append(prefix[-1] + value)

# 計算區間 [l, r) 的和
def range_sum(l, r):
    return prefix[r] - prefix[l]

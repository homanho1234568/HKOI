第四章｜陣列與搜尋策略

能夠快速處理串列、理解線性搜尋與二分搜尋，是應對絕大多數資料處理題的核心技能。

陣列／串列常用技巧

  • 初始化：Python 可用 [0] * n 建立固定長度；C++ 使用 vector a(n)。
  • 遍歷：同時需要索引與值時，Python 可用 for idx, val in enumerate(arr)。
  • 切片與複製：arr[a:b] 會建立新串列，考試中避免不必要的複製以節省時間。
# 篩選大於平均值的元素
arr = list(map(int, input().split()))
avg = sum(arr) / len(arr)
filtered = [x for x in arr if x > avg]
print(len(filtered), filtered)

線性搜尋與早停

線性搜尋簡單可靠，若搭配早停可以減少不必要的比較。

target = int(input())
arr = list(map(int, input().split()))

found = -1
for i, val in enumerate(arr):
    if val == target:
        found = i
        break

print(found)
Tip：輸出 -1 表示未找到是 HKOI 常見格式，務必與題目規定一致。

二分搜尋與前置條件

二分搜尋的重點是資料必須已排序，可用 Python 的 bisect 或自行實作。

import bisect

arr = sorted(map(int, input().split()))
x = int(input())

pos = bisect.bisect_left(arr, x)
if pos < len(arr) and arr[pos] == x:
    print("Found at", pos)
else:
    print("Not found")

若需要實作 C++ 版本，可使用 std::lower_bound；計算上下界時記得處理「找不到」的情況。

練習建議

  1. 輸入 N 與 N 個整數，輸出所有不同的數並依照出現頻率排序。
  2. 給定已排序的整數串列與多個查詢，回傳每個查詢值的出現次數。
  3. 模擬排序後的搜尋：先將原始資料排序，再找出最接近的兩個數字。

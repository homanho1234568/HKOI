第一章｜變數（Variables）與算式（Expressions）

程式的生命週期：讀入 → 處理 → 輸出。打好這個基本功，HKOI 多數題目就能迎刃而解。

變數是什麼？

把變數想成貼有標籤的箱子。箱子裡放的是資料，標籤則是名稱。Python 不需要事先宣告資料型別，直接賦值即可。

# 建立變數並放入不同資料
a = 10              # 整數
name = "HKOI"       # 字串
pi = 3.14159        # 浮點數
is_learning = True  # 布林值
命名技巧：使用有意義的英文單字，並以小寫與底線組成，例如 total_score。Python 變數不能以數字開頭。

算式與輸入輸出

大部分 HKOI 題目都會要求你：

  1. 使用 input() 讀入資料（預設為字串）。
  2. 在算式中運用變數進行處理。
  3. 用 print() 按題目格式輸出。
# 經典 A + B 題
a = int(input())
b = int(input())
print(a + b)
注意：input() 回傳值是字串。若要做加減乘除，請先用 int() 或 float() 轉型。

畢氏定理範例：歐幾里得距離

距離公式：c = √(a² + b²)。Python 用 ** 代表次方，用 **0.5 相當於開平方根。

a = 3
b = 4
c = (a ** 2 + b ** 2) ** 0.5
print(c)  # 5.0

這類題目常見於計算平面距離或幾何判斷。

掌握基本算術與運算子

算術運算

  • + 加法、- 減法、* 乘法、/ 浮點除法。
  • // 整除，% 取餘數，用於同餘與循環。
  • ** 次方，例如 2 ** 5 == 32。

比較與布林運算

  • == 等於、!= 不等於、> 與 < 比較大小。
  • and、or、not 控制條件邏輯。
  • 多條件時可以搭配括號確保判斷順序。

練習：拆解輸入輸出題

  1. 題目給出兩個整數和一個字串，要求輸出字串重複兩次及兩數相加結果。提示：用 int() 轉型，字串可用 * 重複。
  2. 輸入三個浮點數 a, b, c，輸出等差平均值與等比平均值。提示：等比平均為 (a * b * c) ** (1/3)。
